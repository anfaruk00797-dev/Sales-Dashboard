#  Sales Dashboard (Streamlit)

Interactive sales dashboard built with **Streamlit**, **Pandas**, and **Plotly**.

This project visualizes sales performance across multiple years and product categories, providing interactive filtering and data insights.

---

##  Features

-  Filter by Year
-  Filter by Product Category
-  Monthly Sales Trend (Line Chart)
-  Sales by Category (Bar Chart)
-  Category Distribution (Pie Chart)
-  Yearly Sales Comparison
-  KPI Metrics (Total, Average, Highest Month)
-  Download Filtered Data (CSV Export)
-  Optimized with Streamlit Caching

---

## 🛠 Tech Stack

- Python 3
- Streamlit
- Pandas
- NumPy
- Plotly Express

## ▶️ How to Run

1. Clone the repository:


git clone <your-repo-link>


2. Install dependencies:


pip install -r requirements.txt


3. Run the app:


streamlit run app.py
