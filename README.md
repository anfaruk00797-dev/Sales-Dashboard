# 💰 Sales Dashboard

A simple interactive Sales Dashboard built with Streamlit.

## 📊 Features

- 3 Interactive Graphs:
  - Line Chart (Monthly Sales)
  - Bar Chart (Sales by Category)
  - Pie Chart (Category Distribution)
- Sidebar Filters (Year & Category)
- Summary Metrics
- Download Filtered Data as CSV

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

---

## 🚀 How to Run

### 1️⃣ Clone Repository

```bash
git clone https://github.com/anfaruk00797-dev/Sales-Dashboard.git
cd Sales-Dashboard
```

### 2️⃣ Create Virtual Environment (Optional)

```bash
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the App

```bash
streamlit run app.py
```
